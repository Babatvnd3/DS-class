Python Function Exercises (Simple "Apps") 
Temperature Converter
Create a function that takes a temperature in Celsius and returns the equivalent temperature in Fahrenheit.


Basic Calculator: Addition
Write a function called add_two_numbers that accepts two numbers as arguments and returns their sum.

Vowel Counter
Develop a function that takes a single word (string) as input and returns the count of vowels (a, e, i, o, u) present in that word.

Greeting Generator
Create a function that takes a user's name and their favorite programming language (both strings) and returns a personalized greeting message.
Example Output: "Hello [Name]! It's great you enjoy learning [Language]."

Area of a Rectangle
Write a function that accepts the length and width of a rectangle and returns its area. The function should assume the inputs are numbers.

Positive/Negative Checker
Design a function that takes a single number and prints whether the number is positive, negative, or zero. (This function uses an if/elif/else structure and uses print instead of return.)

List Maximum Finder
Create a function that takes a list of numbers (e.g., [10, 5, 25, 8]) and returns the largest number in that list.

String Reverser
Write a function that accepts a string and returns the reversed version of that string.


Simple Password Validator
Develop a function that takes a password string and returns True if the password is at least 8 characters long, and False otherwise.

Discount Calculator
Create a function that takes an item's original price and a discount percentage (e.g., 10 for 10%) and returns the final price after applying the discount.
